## DEZD

Suggested slug: `/dictionary-source/dezd/`

Suggested attribution text: `Data from DEZD - Kirmancki Dictionary (CC BY 4.0).`

Suggested page body:

> **DEZD - Kirmancki Dictionary**
>
> DEZD is a digital Kirmancki dictionary source used in the LL Tools dictionary on this site. The public dataset we worked from is published as **Manidar/DEZD-Dictionary-Kirmancki**.
>
> The dataset description says DEZD is part of the **DEZD Language Dataset** and describes it as a resource for the scientific documentation and digital preservation of **Kirmancki / Zonê Ma / Zazaki / Dimili**. The same description says the project was realized by **Inisiatifê Projea Zonê Ma** together with the online dictionary **qesebend-sozluk.de**.
>
> On this site, DEZD is used as one of the dictionary sources for headwords, glosses, and related forms. Individual entries may also combine evidence from other dictionaries, so the source badge on each entry shows which dictionary or dictionaries contributed to that entry.
>

> **Palu - Bingöl Harun Turgut**
>
> This source is based on **Harun Turgut, _Zazaca-Türkçe Sözlük_**, published by **Vejiyaişê Tiji / Tij Yayınları**. The local copy used in the LL Tools import workflow identifies the **second edition (2006)** and also notes an earlier **first edition (2001)**.
>
> In the front matter, the dictionary is described as a Zazaca-Turkish dictionary based on **Zazaca spoken in Palu, Bingöl, and surrounding areas**. In the preface to the second edition, Harun Turgut explains that the new edition was intended to correct many printing errors and other gaps from the first edition, while also making clear that the work should not be treated as a final or exhaustive dictionary of the language.
>
> On this site, Harun Turgut is used as one of the dictionary sources for headwords, glosses, and related forms. When an entry combines material from more than one dictionary, the source badges on the entry show which dictionaries contributed to that entry.
